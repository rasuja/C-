# C# 入門課題：Challenge 8「データの削除：タスクを消去する」

Webアプリには「削除」機能も欠かせません。
今回は、リストの中から特定のタスクを選んで、リストから完全に削除する処理に挑戦しましょう。

## 課題内容
登録したタスクの中から1つ選び、それをリストから削除して再表示してください。

1.  **事前準備**: Challenge 7 と同様に、いくつかタスクをリストに入れておきます。
2.  **削除前の表示**: 番号（添字）付きで全表示します。
3.  **削除処理**:
    - `Console.ReadLine()` で「何番を削除しますか？」と聞き、番号を入力してもらいます。
    - リストの `.RemoveAt(index)` メソッドを使って、その番号の要素を削除します。
4.  **後の表示**: リストを再表示し、指定したタスクが消えていることを確認します。

### 期待される出力例
```text
--- 現在のタスク ---
[0] 買い物
[1] 掃除

削除する番号を入力してください：
> 0

--- 削除後のタスク ---
[0] 掃除
```

## ヒント
- **削除のコード**:
```csharp
int index = int.Parse(Console.ReadLine());
task.RemoveAt(index); // 指定した番号の要素を消す
```

## 進め方
1.  `Program.cs` を書き換えて挑戦してください。
2.  削除後にリストの番号が自動で詰められる（1番だったのが0番になる）様子を観察してみてください！
